## 👨‍💻 ประวัติโดยสรุป (Professional Summary)

.NET Developer ที่มีประสบการณ์ทำงานจริง 1 ปี ณ **สำนักนวัตกรรมดิจิทัลและระบบอัจฉริยะ (PSU DIIS)** มีความเชี่ยวชาญสูงในการสร้าง **RESTful API** ที่แข็งแกร่ง (HttpClient, Swagger) และการออกแบบโครงสร้างฐานข้อมูลที่มีประสิทธิภาพเพื่อแก้ปัญหาโจทย์ที่ซับซ้อน

**ความสามารถหลัก (Backend Focused):**
- **API Development:** ออกแบบและพัฒนา Backend API ด้วย **ASP.NET Core** และ **Node.js**
- **Database Architecture:** เชี่ยวชาญการออกแบบ Schema ทั้ง **Relational (PostgreSQL)** และ **Graph Database (Neo4j)** - **Enterprise Solutions:** พัฒนาระบบจัดการเอกสารอิเล็กทรอนิกส์ (EDOC) และระบบบริหารจัดการทรัพยากรบุคคล (HRMS)
- **Business Logic:** ออกแบบ Service สำหรับประมวลผลข้อมูลร่วมกับ **Blazor** และ **.NET 8** ---

### 📦 [Asset Management System](https://github.com/PanuwatBuapetch/Asset_Management_System)
**บทบาท:** Backend Developer (System Logic & Database)

ระบบบริหารจัดการครุภัณฑ์และอุปกรณ์ภายในองค์กร เน้นความถูกต้องของการไหลของข้อมูลและการจัดการสถานะทรัพยากร

- **⚙️ Core Service:** พัฒนาส่วนควบคุม Logic สำหรับการบันทึก เบิกจ่าย และติดตามสถานะอุปกรณ์ (Asset Tracking)
- **💾 Data Integrity:** ออกแบบโครงสร้างฐานข้อมูลเพื่อรองรับประวัติการใช้งานและข้อมูลทางสถิติของครุภัณฑ์
- **🚀 Efficiency:** ปรับจูนระบบการรับส่งข้อมูลเพื่อให้สามารถจัดการรายการอุปกรณ์จำนวนมากได้อย่างรวดเร็ว

### 🧩 [Modern HRMS PRO (.NET 8)](https://github.com/PanuwatBuapetch/HRMSProject)
**บทบาท:** Backend Developer (System Architecture & API)

ระบบบริหารจัดการทรัพยากรบุคคลระดับองค์กรที่เน้นความปลอดภัยและโครงสร้างข้อมูลที่ซับซ้อน

- **🛡️ Advanced Security:** พัฒนาระบบ Authentication พร้อมระบบแยกสิทธิ์เข้าถึงข้อมูล (Role-based Access Control)
- **🌍 Dynamic Localization:** ออกแบบระบบสลับภาษา ไทย-อังกฤษ จากฝั่ง Server เพื่อรองรับมาตรฐานสากล
- **💾 Database Architecture:** ออกแบบ Schema บน **PostgreSQL** รองรับโครงสร้างหน่วยงานและตำแหน่งงานที่ซับซ้อน
